## Prepare

> install git

``` sh
sudo apt-get install git
```




### Clone

> clone [ubuntu-iso-builder-respin-xfce](https://github.com/samwhelp/ubuntu-iso-builder-respin-xfce)

``` sh
git clone https://github.com/samwhelp/ubuntu-iso-builder-respin-xfce.git
```


> change dir to `ubuntu-iso-builder-respin-xfce`

``` sh
cd ubuntu-iso-builder-respin-xfce
```




### Build

> bulid

``` sh
make build
```

> The resulting ISO file will be placed on `ubuntu-iso-builder-remix-xfce/dist`
